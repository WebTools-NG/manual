Find Media module

Main Menu

The Find Media module allows you to scan your local folders and compare the list to the files held in your Plex Media Server database in order to locate files that may exist locally, but have not been found or included in your Plex Media Server library.

Note: In order to use this module, you must be local to your Plex Media Server media storage device.

Before proceeding, please see the Find Media Settings page and the Global Settings > Library Mapping page to make sure all settings are correct.

Before you proceed, make sure you have selected a server to work with. Most operations will fail if you have not selected a server.

Click on the Select Media Library drop down menu and select the library you wish to work with.

Note: Make sure you have already selected the correct library to work with on the Global Settings > Library Mapping page and configured the folder path correctly, if needed.

Once the library has been selected above, click on the Run button to find any missing media.

The progress will be displayed in the Status: box.

Check the .csv and/or .xlsx file to see the list of missing files.
